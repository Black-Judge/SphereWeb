Добавление нового шаблона демо страницы

Прежде всего, Я прошу Вас отнестись крайне важно к моим рекомендациям:

  1. Используйте главную страницу исключительно как демонстративную.
  2. Не используйте дизайны сайтов где background'ом является весомые видео.
  3. Не используйте никакие всплывающие окна.
  4. Ваша демо страница должна содержать только актуальную информацию, а устаревшая информация - удалена.
Демо страница - главная страница, как правило, состоящий из одной (или нескольких) веб-страниц и содержащий основную информацию.

Вы можете создать свою демо страницу, либо использовать готовую.
Для чтоб создать свою демо страницу, Вам необходимо создать директорию по адресу ~/template/designs/, назовите её и создайте там файл index.html.
index.html - это и есть тот файл, который будет загружаться при входе пользователя на главную страницу Вашего сайта, разместите в этом файле необходимую информацию о Вашем сервере, ссылки на новости, страницы, статистику и т.д.

Для того чтоб переместить уже готовый шаблон в нашу систему, Вам необходимо разместить шаблон по адресу ~/template/designs/, в котором главный файл должен называться index.html.
Основной шаблонизатора является TWIG, ознакомьтесь с его синтаксисом.
Вы можете использовать множество дополнительных псевдопеременных, которые предлагает система.
