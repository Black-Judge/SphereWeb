Регистрация игрового сервера

Чтоб совместить работу сайта и Вашего сервера, Вам необходимо зарегистрировать в системе информацию о Вашем сервере.
Регистрация сервера находится в меню Админ-Панель -> Настройки -> Добавить.

Информация о сервере необходима, для того чтоб информировать пользователя с каким сервером он взаимодействует.
