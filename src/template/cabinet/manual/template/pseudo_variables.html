Шаблонные псевдопеременные и функции

По мимо использования обычных функций TWIG, вы можете использовать зарезервированные системой псевдопеременные на демо страницы.

В некоторых функциях, нужно передавать параметры, для обозначения типа(ов) параметров используется следующие обозначения.
int - целочисленное значение
string - числовое значение
| - или, к примеру (string|int) означает что допустимо вводить либо строку в кавычках, либо число.
... - перечисление N значений

Псевдопеременные
Вызов Описание
phrase(int|string) Возвращает языковую фразу по ключу либо строчному идентификатору.
get_template() Название действующего демо шаблона
format_number_fr(int) Аналог функции number_format, функция принимает число, приведение числа в более человекоразличимое значение, используя пробелы через каждые 3 символа.
timeHasPassed(int) Функция должна получить N секунд. Возвращает результат преобразования секунд во временный период (сек, мин, часы, дни, года).
file_exists(string) Функция принимает адрес файла, и возвращает bool (true/false) существования файла.
get_youtube_id(string) Функция принимает адрес youtube видео, и возвращает ID ролика.
lang_flag() Возвращает строку с установленным у пользователя в данный момент языкового интерфейса (ru или gb). Используется для выводов изображений флагов.
last_forum_message(int) Функция принимает N значение последних сообщений, которые будут выводиться с форума на движке xenforo.
get_forum_link(int, int) Генерирует ссылку перехода к последнему сообщению.
Функция принимает два значения:
1. thread_id - ID темы.
2. post_id - ID поста.
forum_enable() Возвращает bool (true/false) - включено ли вывод сообщений с форума.
forum_user_avatar(int) Возвращает string ссылку на аватар пользователя форума. Функции необходимо передать ID пользователя.
get_enable_game_chat() Возвращает bool (true/false) - включено ли функция вывода на сайте игрового чата.
get_item_info(int) Возвращает краткую информация об игровом предмете.
get_screen_enable() Возвращает bool (true/false) - включена ли опция включения загрузки пользователями скриншотов.
grade_img(int) Возвращает string ссылку на иконку класса предмета. Функция принимает класс предмета (crystal_type).
castle_name(int) Возвращает название замка. Функция принимает ID замка.
user_info(string) Функция принимает строковое значение, которое определенно к какому-то типу информации о пользовательском аккаунте.
Типы информации, которые принимает функция user_info
get_default_server - Возвращаем ID сервер, который установлен по умолчанию у пользователя.
get_id - ID пользователя.
get_is_auth - Является ли пользователь авторизированным .
get_email - E-Mail пользователя.
get_password - Пароль пользователя.
get_name - Имя пользователя.
get_ip_registration - IP при регистрации.
get_ip - IP пользователя.
get_date_create - Дата регистрации.
get_date_update - Дата последнего изменения профиля.
get_access_level - Возвращает статус пользователя (guest,user,moderator,admin).
get_donate_point - Количество донат очков. Является валютой во внутреннем кабинете.
get_class(int) Функция получает ID класса персонажа, и возвращает название профессии.
get_class_race(int) Функция получает ID расы персонажа, и возвращает название расы.
sex(int) Функция принимает id пола персонажа, и возвращает либо male(мужчина) либо female(женщина).
