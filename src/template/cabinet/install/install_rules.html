Sphere-Web является бесплатной веб разработкой для Java серверов написана на языке PHP.
В котором реализован огромный функционал для администрирования сайта и личного кабинета.

  1. Запрещено продавать CMS или CMS где используется код Sphere-Web
  2. Запрещено предоставлять платные услуги в данной CMS
    1. За исключением: Работа над веб-шаблонами
    2. За исключением: Принимающим участие в разработке Sphere-Web
  3. Копии Sphere-Web должны содержать ссылки на оригинал github проекта Sphere-Web

Вы можете заказать разработку дополнительного функционала у разработчика CMS - мой Telegram.
