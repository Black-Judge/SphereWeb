О загрузчике Sphere-Launcher
Sphere Launcher - это программа для загрузки файлов игры, запуска. С большим набором функционала.
Главные задачи при разработки лаунчера были:
  1. Максимальная скорость работы, выжать из ПК максимальную скорость загрузки, обработки.
  2. Отказ от повторных загрузок файлов, если пользователь их загружал. Так как 87% файлов в патчах, это файлы-дубликаты, которые раньше или постоянно игрок загружает.
  3. Создание универсального лаунчера, который можно использовать игроком для всех серверов.
  4. Поддержка автологина в игру (для патчей, которые это подддерживают).
  5. Кастоматизация и простое изменение интерфейса, достаточно знать базово HTML / CSS.
Думаю в большей мере я справился с задачей, огромное сложности были преодолены и продолжаю улучшать, расширять возможности и решать выявленные ошибки. По этому лаунчер имеет функцию автообновления.
Как добавить свой лаунчер
Для того чтоб сделать патч, необходимо выполнить следующие действия.
  1. Сгенерировать патч.
  2. Залить архивы себе на сайт / сервер.
  3. Разработчику лаунчера передать ссылки на файл с архивом и патч листом.
  4. Получить от разработчика tokenApi.
  5. Добавить информацию о Вашем лаунчере.
Сколько можно создавать лаунчеров?
Вы можете создавать неограниченное кол-во лаунчеров под свой сервер.
Что такое токен и зачем он нужен

Токен это публичный ключ, одна из систем безопасности.
Каждый токен, имеет струю привязку к патчу Вашему серверу.

Что будет если другой сервер будет использовать мой токен?

Если кто-то решит использовать Ваш токен для своего лаунчера, тогда игроки сервера-воришки скачают Ваш патч и зайдут на Ваш сервер.

Как работает данный лаунчер

Лаунчер работает через веб-браузер, это есть Ваш интерфейс лаунчера.
Посетителю на странице лаунчера, будет предложено скачать или запустить лаунчер.
Если игрок ранее загружал уже лаунчер, вне зависимости с какого источника (всегда загружается с основного репозитория), тогда он сможет просто запустить лаунчер из браузера. Игроку нет необходимости искать ярлык лаунчера или загружать повторно.
После запуска лаунчера, пользователь сможет воспользоваться всем функционалом.
